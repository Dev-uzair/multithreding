Method references in Java are a shorthand notation for writing lambda expressions.
There are four types of method references:

Static Method Reference:

Syntax: ClassName::staticMethodName
Example: Math::max
Used when: Referring to static methods


Instance Method Reference of Particular Object:

Syntax: objectInstance::instanceMethodName
Example: System.out::println
Used when: Calling instance method on a specific object


Constructor Reference:

Syntax: ClassName::new
Example: ArrayList::new
Used when: Creating new instances
